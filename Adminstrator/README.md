# Meeting Invitation System

A React frontend for managing meeting invitations, viewing participants, generating read-only invitation forms, and tracking sent invitation logs.

## Requirements

- Node.js 18+
- npm

## Install

```bash
npm install
```

## Run

```bash
npm run dev
```

Open the local URL printed by Vite, usually `http://localhost:5173`.

## Routes

- `/` - Meeting List
- `/meeting/:id` - Participant List
- `/logs` - Log Book

## Tech Stack

- React functional components and hooks
- React Router DOM
- React Context for app state
- Plain CSS
